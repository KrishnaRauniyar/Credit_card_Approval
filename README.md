# Credit_card_Approval
The data set is taken from https://archive.ics.uci.edu/ml/datasets/default+of+credit+card+clients. You can still download from my repository
above. The dataset contains of 16 columns and about 690 instances. The detailed information of the data is given in this site 
http://rstudio-pubs-static.s3.amazonaws.com/73039_9946de135c0a49daa7a0a9eda4a67a72.html. 
I have used linear regression with all the default parameters and got about 84% accuracy_score. 
Next I used GridSearch algorithm with cross validation of 5 to best hyper tune the model and the accuracy increased.
